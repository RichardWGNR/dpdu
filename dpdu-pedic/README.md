# DPDU pEDIC

## Open-source DPDU driver implementation for ISO 11898 RAW CAN communication

dpdu-pedic is an alternative DPDU implementation designed to replace the proprietary Softing EDIC driver for applications requiring low-level access to ISO 11898 RAW communication.

The project provides an open and extensible driver layer for automotive diagnostics, CAN development tools, and test environments where direct RAW CAN frame exchange is required without dependency on proprietary vendor software.
